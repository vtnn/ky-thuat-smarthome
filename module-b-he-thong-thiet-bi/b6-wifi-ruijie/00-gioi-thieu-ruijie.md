# B6.00 — Giới Thiệu WiFi Ruijie
Ruijie là hãng thiết bị mạng — AP, Switch, Controller. Sử dụng cho giải pháp WiFi phủ sóng toàn nhà/tòa nhà.
## Kiến trúc
```
[Internet/Modem] → [Router] → [Switch PoE Ruijie]
                                    ├── [AP #1 - Tầng 1]
                                    ├── [AP #2 - Tầng 2]
                                    └── [AP #3 - Sân vườn]
                   [Ruijie Cloud Controller] (quản lý tập trung)
```
