# B5.06 — Remote Access (Hik-Connect)

## Hik-Connect
- Dịch vụ Cloud P2P của Hikvision — xem camera từ xa **không cần port forwarding**.
- App: Hik-Connect (iOS/Android).

## Cấu hình
1. NVR → Configuration → Network → Platform Access → Enable Hik-Connect.
2. Quét mã QR hoặc nhập serial NVR trên App.
3. Đăng nhập tài khoản Hik-Connect → thêm thiết bị.
4. Xem live, playback, nhận push notification.

## Chia sẻ cho khách hàng
- Tạo tài khoản Hik-Connect cho khách → chia sẻ thiết bị.
- Hoặc bàn giao tài khoản chính sau quyết toán.

## Lưu ý bảo mật
- Bật mã xác minh (verification code) trên NVR.
- Không dùng mật khẩu đơn giản cho tài khoản Hik-Connect.
