# Module Planner

Use this skill when opening a new module or reorganizing a lesson sequence.

## Produce
- module goal
- lesson list
- recommended lesson order
- required examples
- likely missing pieces
- done criteria for each lesson

## Planning rules
- Start from technician learning needs
- Keep progression from basic to applied
- Ensure each module closes the loop: identify -> install -> configure -> troubleshoot
- Avoid creating modules that are broad but shallow
