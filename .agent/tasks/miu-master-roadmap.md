# Miu Master Roadmap — ky-thuat-smarthome

## Mục tiêu
Biến repo thành wiki kỹ thuật dùng thật cho đào tạo, tra cứu và vận hành nội bộ.

## Ưu tiên tổng thể
1. Khóa module A + B1
2. Hoàn thiện B2
3. Mở B3/B4/B5/B6
4. Chuẩn hóa toàn repo

## Done criteria chung
- Có title, description, module
- Có mục tiêu đầu bài
- Không H1 trong body
- Giọng văn tự nhiên, thực chiến
- Có ví dụ thật, lỗi thường gặp, checklist nếu phù hợp
- Build pass, link đúng, assets đúng
