## Mục tiêu
- Phân biệt nhanh thiết bị **nguồn bus**, **actuator**, **gateway** và **thiết bị đầu cuối**.
- Biết hãng phổ biến để đối chiếu khi chọn vật tư/nhập catalog vào ETS.

---

## 1. Danh mục thiết bị KNX

| Thiết bị | Chức năng | Hãng phổ biến |
|---|---|---|
| KNX Power Supply 320/640mA | Cấp nguồn 29V DC cho bus | ABB, Schneider |
| Push Button Interface 1/2/4/8 nút | Nút bấm thông minh | ABB, HDL, Zennio |
| Switch Actuator 4/8/12 kênh | Relay bật/tắt (6–16A/kênh) | ABB, Schneider |
| Dimming Actuator 2/4 kênh | Dim đèn LED/halogen | ABB, Schneider |
| Blind/Shutter Actuator 2/4 kênh | Điều khiển motor rèm | ABB, Schneider |
| Binary Input 4/8 kênh | Dry contact → tín hiệu KNX | ABB, HDL |
| KNX-DALI Gateway | KNX ↔ DALI (64 ballast) | ABB, Tridonic |
| KNX/IP Gateway | KNX ↔ Ethernet | ABB, Weinzierl |
| USB Interface | Kết nối ETS qua USB | ABB, Weinzierl |
| Bus Cable LIYCY 2×2×0.8 | Cáp bus xanh lá | Chuẩn KNX |
