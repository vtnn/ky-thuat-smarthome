# Kỹ thuật Smarthome

Chào mừng anh em tới wiki kỹ thuật. Nội dung được quản lý bằng Markdown trong repo GitHub để cập nhật nhanh, review bằng PR, và deploy tự động lên Cloudflare Pages.

## Bắt đầu nhanh
- Vào **Tổng quan**: xem cấu trúc tài liệu và cách học theo level.
- Duyệt theo **Modules** (A → G) ở sidebar.
- Xem **FAQ / Glossary** trong mục **Reference**.

> Góp ý/chỉnh sửa: tạo PR hoặc edit trực tiếp file `.md` trên GitHub.
