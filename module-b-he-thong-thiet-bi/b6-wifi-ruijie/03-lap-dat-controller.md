# B6.03 — Lắp Đặt Ruijie Controller
## Cloud Controller
- Quản lý tập trung tất cả AP qua cloud hoặc local.
- Cấu hình SSID, VLAN, firmware, giám sát.
## Cấu hình
1. Đăng ký tài khoản Ruijie Cloud.
2. Thêm AP bằng serial/MAC.
3. Tạo SSID theo quy hoạch VLAN.
4. Đẩy cấu hình xuống tất cả AP.
