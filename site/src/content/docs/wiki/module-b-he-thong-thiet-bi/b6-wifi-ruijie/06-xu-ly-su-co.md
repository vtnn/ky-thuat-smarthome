# B6.06 — Xử Lý Sự Cố WiFi

| Triệu chứng | Kiểm tra |
|-------------|----------|
| WiFi chậm | Băng tần, số thiết bị, kênh, nhiễu |
| Không kết nối | Mật khẩu, SSID, VLAN, DHCP |
| Mất kết nối gián đoạn | AP quá tải, roaming, driver |
| IoT mất kết nối | 2.4GHz, SSID riêng, IP conflict |
| Vùng chết | Thiếu AP, vị trí AP sai |

## Checklist
- [ ] AP có nguồn PoE? LED sáng?
- [ ] SSID đúng VLAN?
- [ ] DHCP đang hoạt động?
- [ ] RSSI ≥ -65dBm tại vị trí lỗi?
- [ ] Kênh WiFi không bị nhiễu?
