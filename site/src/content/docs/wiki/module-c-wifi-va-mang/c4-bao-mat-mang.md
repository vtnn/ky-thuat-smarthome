# C4 — Bảo Mật Mạng
## Checklist bảo mật
- [ ] Đổi password mặc định TẤT CẢ thiết bị (Router, Switch, NVR, Camera, Hub).
- [ ] Tắt services không dùng (UPnP, Telnet, HTTP không mã hóa).
- [ ] Enable HTTPS/SSH (disable HTTP/Telnet).
- [ ] Cập nhật firmware định kỳ (hàng tháng kiểm tra, hàng quý cập nhật).
- [ ] WiFi: WPA2/WPA3, mật khẩu ≥ 12 ký tự.
- [ ] Kiểm tra thiết bị lạ kết nối mạng hàng tuần.
